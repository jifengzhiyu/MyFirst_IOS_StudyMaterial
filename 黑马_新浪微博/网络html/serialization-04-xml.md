# XML 介绍

* `XML`的特点，出身名门，`W3C`制定，微软和IBM曾经共同大力推荐过的数据格式
* XML 指可扩展标记语言（eXtensible Markup Language）
* 被设计用来传输和存储数据
* HTML 是设计用来表示页面的

## XML 解析的方式

### DOM 解析

- 是在 MAC 使用的解析方式
- **内存消耗极大，不适用于手机**
- iPhone无法直接使用 DOM 方式解析 XML

### SAX 解析

- 是只读的方式，从上向下的方式解析
- 是苹果提供的解析方式
- 速度快
- NSXMLParser 通过`代理`实现解析

## SAX 解析步骤

1. 开始文档 - 准备工作
2. 开始"节点"
3. 发现节点内部的内容，每一个节点，可能需要多次才能找完
4. 结束"节点"
5. 结束文档 - 解析结束

> 以上步骤，2，3，4，会不断循环，一直到所有解析完成！
